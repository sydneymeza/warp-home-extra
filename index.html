Extra Credit

From our meeting with the company, we realized that we were still missing some key navigation elements for the team page. We switched the button popups to tabs for members, billing, and settings. We also changed the function of our trash icon, as we got feedback that many people would probably understand it as a button to go to the trash page, rather than being able to click and drag something to put it in the trash.

The order of content on the beginner page was also changed. From the feedback it was understood that beginners should have the videos display first on their screen (instead of last), and as the users use Warp more, the home page would dynamically change.

A couple more small changes include labeling “Activity” as “Recent Activity” and providing a link for users to view more activity in a new page if what’s provided on the sidebar is too little.
